```

                    ┌─────────────┐

                    └──────┬──────┘
                           │
                    ┌──────▼──────┐
                    │  bootstrap  │

                    └──────┬──────┘
```

## 数据流

```
HTTP Request
     │
     ▼
┌────────────┐
│ Middleware │ ──▶ RequestID / Trace / Metrics / CORS / Logger / Recovery
└─────┬──────┘
      │
      ▼
┌────────────┐
│   Router   │ ──▶ 路由匹配 (public / secured / currentUser)
└─────┬──────┘
      │
      ▼
┌────────────┐
│  Handler   │ ──▶ 参数绑定 (BindJSON / BindQuery)
└─────┬──────┘
      │
      ▼
┌────────────┐
│  Service   │ ──▶ 业务逻辑 / 缓存读写 / 事务管理
└─────┬──────┘
      │
      ▼
┌────────────┐
│ Repository │ ──▶ GORM 查询 / 事务执行
└─────┬──────┘
      │
      ▼
┌────────────┐
│    DB      │ ──▶ PostgreSQL
└────────────┘
```

## 缓存策略

```
┌─────────────┐         Cache Miss          ┌─────────────┐
│   Client    │ ───────────────────────────▶ │     DB      │
└──────┬──────┘                            └─────────────┘
       │
       │ Cache Hit
       ▼
┌─────────────┐
│    Redis    │
└─────────────┘

Profile Cache Key:  {app}:profile:{project_id}:{user_id}
Post Detail Key:    {app}:post:detail:{project_id}:{post_id}
TTL:                configurable (default 5 mins)
```
